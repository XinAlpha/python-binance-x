## 1️⃣ 均线交叉策略 (MA Crossover)

### 策略原理
使用短期和长期移动平均线的交叉作为买卖信号。

### 交易规则
- **做多**：短期均线上穿长期均线（金叉）
- **做空/平多**：短期均线下穿长期均线（死叉）
- **止损**：固定百分比止损
- **止盈**：固定百分比止盈

### 适用场景
✅ 趋势明显的市场
✅ 波动较大的币种
❌ 横盘震荡市场

### 参数配置
```yaml
strategy:
  name: "ma_crossover"
  params:
    ma_short: 10          # 短期均线周期
    ma_long: 30           # 长期均线周期
    stop_loss: 0.02       # 止损2%
    take_profit: 0.04     # 止盈4%
```

### 优缺点
**优点**：
- 简单易懂，容易实现
- 能捕捉大趋势
- 适合新手学习

**缺点**：
- 震荡市场会频繁假突破
- 有滞后性
- 止损可能较大

### 优化建议
- 添加成交量确认
- 结合趋势强度指标（ADX）
- 使用EMA代替SMA减少滞后

## 2️⃣ RSI超买超卖策略

### 策略原理
利用RSI指标识别超买超卖区域，在极值区域反向交易。

### 交易规则
- **做多**：RSI < 30（超卖）且开始回升
- **做空**：RSI > 70（超买）且开始回落
- **平多**：RSI >= 70 或显著下跌
- **平空**：RSI <= 30 或显著上涨

### 适用场景
✅ 震荡市场
✅ 区间交易
❌ 强趋势市场（会过早入场）

### 参数配置
```yaml
strategy:
  name: "rsi"
  params:
    rsi_period: 14        # RSI周期
    oversold: 30          # 超卖阈值
    overbought: 70        # 超买阈值
    use_divergence: false # 是否使用背离
    stop_loss: 0.02
    take_profit: 0.03
```

### 优缺点
**优点**：
- 在震荡市捕捉反转
- 胜率较高
- 风险相对较低

**缺点**：
- 趋势市场会亏损
- 可能过早入场
- 需要及时止损

### 优化建议
- 结合背离信号提高准确性
- 添加趋势过滤器
- 调整超买超卖阈值（如25/75）

## 3️⃣ 布林带突破策略

### 策略原理
利用布林带的上下轨判断价格波动范围，在突破或回归时交易。

### 两种模式

#### 模式1：均值回归（默认）
- **做多**：价格触及下轨后反弹
- **做空**：价格触及上轨后回落
- **平仓**：价格回到中轨

#### 模式2：趋势跟随
- **做多**：价格突破上轨
- **做空**：价格跌破下轨
- **平仓**：价格回到中轨

### 适用场景
✅ 适应多种市场环境
✅ 波动率分析
✅ 突破交易

### 参数配置
```yaml
strategy:
  name: "bollinger_bands"
  params:
    bb_period: 20             # 布林带周期
    bb_std: 2.0               # 标准差倍数
    mode: "mean_reversion"    # 模式：mean_reversion 或 trend
    squeeze_threshold: 0.02   # 挤压阈值
    stop_loss: 0.025
    take_profit: 0.04
```

### 优缺点
**优点**：
- 适应性强
- 能识别波动率变化
- 布林带挤压预示突破

**缺点**：
- 参数敏感
- 需要选择正确模式
- 假突破风险

### 优化建议
- 使用ATR动态调整带宽
- 结合成交量确认突破
- 在挤压后等待方向明确

## 4️⃣ 网格交易策略

### 策略原理
在价格区间内设置多个网格线，价格每触及一个网格就交易，低买高卖。

### 交易规则
- **设置网格**：在价格区间内均匀分布N个网格
- **买入**：价格下跌到网格线
- **卖出**：价格上涨到网格线
- **持续循环**：不断低买高卖

### 适用场景
✅ 横盘震荡市场（最佳）
✅ 高波动率币种
❌ 单边强趋势（会持续亏损）

### 参数配置
```yaml
strategy:
  name: "grid_trading"
  params:
    grid_num: 10              # 网格数量
    price_range: 0.2          # 价格区间±20%
    grid_mode: "arithmetic"   # 网格模式：arithmetic 或 geometric
    rebalance_threshold: 0.3  # 重置网格阈值
    stop_loss: 0.05           # 整体止损
```

### 优缺点
**优点**：
- 震荡市胜率极高
- 持续盈利
- 适合量化交易

**缺点**：
- 趋势市会大幅亏损
- 需要较多资金
- 需要频繁交易

### 优化建议
- 动态调整网格区间
- 设置整体止损
- 结合趋势判断暂停网格

## 5️⃣ MACD趋势策略

### 策略原理
使用MACD的快慢线交叉和柱状图判断趋势强度和转折点。

### 交易规则
- **做多**：MACD金叉（快线上穿慢线）
- **做空**：MACD死叉（快线下穿慢线）
- **确认**：柱状图方向一致
- **强信号**：零轴突破

### 适用场景
✅ 趋势市场
✅ 趋势确认
✅ 背离分析

### 参数配置
```yaml
strategy:
  name: "macd"
  params:
    fast_period: 12           # 快线周期
    slow_period: 26           # 慢线周期
    signal_period: 9          # 信号线周期
    use_histogram: true       # 使用柱状图确认
    use_zero_cross: false     # 使用零轴穿越
    stop_loss: 0.025
    take_profit: 0.05
```

### 优缺点
**优点**：
- 趋势跟踪能力强
- 背离信号准确
- 多重确认机制

**缺点**：
- 滞后性明显
- 震荡市假信号多
- 需要较长K线

### 优化建议
- 结合零轴位置判断主趋势
- 使用背离提前预警
- 柱状图快速衰减时提前出场

## 6️⃣ 突破回踩策略

### 策略原理
等待价格突破关键支撑/阻力位后，在回踩确认时入场，提高胜率。

### 交易规则
- **步骤1**：识别关键支撑阻力位
- **步骤2**：等待突破（成交量确认）
- **步骤3**：等待回踩到关键位
- **步骤4**：确认反转后入场

### 适用场景
✅ 趋势市场
✅ 关键位突破
✅ 盘整后方向选择

### 参数配置
```yaml
strategy:
  name: "breakout_pullback"
  params:
    lookback_period: 20       # 回溯周期
    breakout_threshold: 0.01  # 突破阈值1%
    pullback_ratio: 0.5       # 回踩50%
    use_volume: true          # 成交量确认
    volume_multiplier: 1.5    # 成交量1.5倍
    stop_loss: 0.03
    take_profit: 0.06
```

### 优缺点
**优点**：
- 入场点更优
- 胜率较高
- 风险回报比好

**缺点**：
- 可能错过快速突破
- 需要耐心等待
- 实现较复杂

### 优化建议
- 使用斐波那契回撤位
- 多时间框架确认
- 设置回踩超时机制

## 🎯 策略选择指南

### 根据市场环境选择

**趋势市场**：
1. MACD趋势策略
2. 均线交叉策略
3. 突破回踩策略

**震荡市场**：
1. 网格交易策略
2. RSI超买超卖策略
3. 布林带均值回归

**不确定市场**：
1. 布林带策略（自适应）
2. 先观察再选择

### 根据风险偏好选择

**保守型**：
- RSI策略（胜率高，收益稳定）
- 网格策略（震荡市稳赚）

**平衡型**：
- 均线交叉（简单实用）
- 布林带策略（灵活）

**激进型**：
- MACD策略（抓大趋势）
- 突破回踩（高风险高回报）

## 🔄 策略组合建议

### 组合1：趋势+震荡
- 主策略：MACD趋势
- 辅助策略：RSI超买超卖
- 逻辑：趋势市用MACD，震荡市用RSI

### 组合2：突破+网格
- 主策略：突破回踩
- 辅助策略：网格交易
- 逻辑：突破时跟随，震荡时网格

### 组合3：多时间框架
- 大周期：均线判断主趋势
- 小周期：RSI找入场点
- 逻辑：顺大势做小波段

## 📈 回测和优化

### 回测步骤
1. 选择策略
2. 修改config.yaml中的参数
3. 运行回测：`python run_backtest.py`
4. 分析结果
5. 调整参数
6. 重复2-5直到满意

### 参数优化建议

**均线周期**：
- 快速：5/15（激进）
- 中等：10/30（平衡）
- 慢速：20/50（保守）

**RSI阈值**：
- 宽松：20/80（减少交易）
- 标准：30/70（平衡）
- 严格：40/60（增加交易）

**止损止盈**：
- 短线：1%/2%
- 中线：2%/4%
- 长线：5%/10%
